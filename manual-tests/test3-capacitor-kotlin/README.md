# test3-capacitor-kotlin

Capacitor Plugin generated with Kotlin for Android, to check https://github.com/ionic-team/create-capacitor-plugin/pull/122

## Install

To use npm

```bash
npm install test3-capacitor-kotlin
````

To use yarn

```bash
yarn add test3-capacitor-kotlin
```

Sync native files

```bash
npx cap sync
```
