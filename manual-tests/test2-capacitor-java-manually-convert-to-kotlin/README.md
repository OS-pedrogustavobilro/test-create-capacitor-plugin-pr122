# test2-capacitor-java-manually-convert-to-kotlin

Plugin generated with Java that was manually converted to Kotlin in Android Studio

## Install

To use npm

```bash
npm install test2-capacitor-java-manually-convert-to-kotlin
````

To use yarn

```bash
yarn add test2-capacitor-java-manually-convert-to-kotlin
```

Sync native files

```bash
npx cap sync
```
